tyc2-sky2000.csv.gz, hip-sky2000.csv.gz, hipdouble-sky2000.csv.gz
	contains crossidentification of Tycho2/Hipparcos/Hipparcos double with Sky2000 master catalog. 
	It was generated using Topcat with two steps:
	    1) Pair match using position with Sky algorithm and 0.5 arcsec precession
	    2) Result table was filtered with subset expression 'abs(Vmag_1- Vmag_2) < 0.5'
